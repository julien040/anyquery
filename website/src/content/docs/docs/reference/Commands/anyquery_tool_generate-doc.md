Generate the markdown documentation of the CLI

```bash
anyquery tool generate-doc [dir] [flags]
```

### Examples

```bash
# Generate the doc in the docs directory
anyquery tool generate-doc docs
```

### Options

```bash
  -h, --help   help for generate-doc
```
